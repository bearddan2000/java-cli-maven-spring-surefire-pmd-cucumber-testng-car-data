# java-cli-maven-spring-surefire-pmd-cucumber-testng-car-data

## Description
A POC for spring app using testng
and cucumber framework with
surefire and pmd plugins.

## Tech stack
- java
- maven
  - spring
  - testng
  - surefire
  - cucumber
  - pmd

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- pmd report found at bin/target/site

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
